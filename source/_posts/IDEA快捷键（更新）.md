  以下是 **IDEA 全场景高频快捷键分类清单**，覆盖 Windows/Mac 双系统，按「编辑→导航→重构→调试→版本控制→窗口操作」分类，兼顾新手入门与老手高效使用，标注核心优先级（★越多越常用）：

### 一、基础编辑（写代码必备，最高频）

| 功能描述                    | Windows 快捷键       | Mac 快捷键          | 优先级 |
| --------------------------- | -------------------- | ------------------- | ------ |
| **单行注释 / 取消注释**     | **Ctrl + /**         | Cmd + /             | ★★★★★  |
| **多行注释 / 取消注释**     | **Ctrl + Shift + /** | Cmd + Shift + /     | ★★★★☆  |
| **复制当前行到下一行**      | **Ctrl + D**         | Cmd + D             | ★★★★★  |
| 删除当前行                  | Ctrl + Y             | Cmd + Delete        | ★★★★☆  |
| 撤销操作                    | **Ctrl + Z**         | Cmd + Z             | ★★★★★  |
| 重做操作（反撤销）          | Ctrl + Shift + Z     | Cmd + Shift + Z     | ★★★★☆  |
| **格式化代码（统一风格）**  | **Ctrl + Alt + L**   | Cmd + Alt + L       | ★★★★★  |
| 自动补全语句（分号 / 括号） | Ctrl + Shift + Enter | Cmd + Shift + Enter | ★★★★☆  |
| 快速复制粘贴（无格式）      | Ctrl + Shift + V     | Cmd + Shift + V     | ★★★☆☆  |
| 大小写转换                  | Ctrl + Shift + U     | Cmd + Shift + U     | ★★☆☆☆  |
| **批量替换文本**            | **Ctrl + R**         | Cmd + R             | ★★★★☆  |
| 全局搜索文本                | Ctrl + Shift + F     | Cmd + Shift + F     | ★★★★☆  |

### 二、导航与搜索（快速定位代码）

| 功能描述                        | Windows 快捷键             | Mac 快捷键           | 优先级 |
| ------------------------------- | -------------------------- | -------------------- | ------ |
| 全局搜索类 / 文件 / 符号        | Double Shift（双击 Shift） | Double Shift         | ★★★★★  |
| 查找类（仅搜索类名）            | Ctrl + N                   | Cmd + O              | ★★★★☆  |
| 查找文件（仅搜索文件名）        | Ctrl + Shift + N           | Cmd + Shift + O      | ★★★★☆  |
| 跳转到行号                      | Ctrl + G                   | Cmd + G              | ★★★☆☆  |
| 查看当前类的结构（方法 / 属性） | Ctrl + F12                 | Cmd + F12            | ★★★★☆  |
| 跳转到定义（如类 / 方法来源）   | Ctrl + B / Ctrl + 点击     | Cmd + B / Cmd + 点击 | ★★★★★  |
| 查看引用（谁调用了当前方法）    | Ctrl + Alt + F7            | Cmd + Alt + F7       | ★★★☆☆  |
| 切换最近打开的文件              | Ctrl + E                   | Cmd + E              | ★★★★☆  |
| 返回上一个光标位置              | Ctrl + Alt + ←             | Cmd + Alt + ←        | ★★★★☆  |
| 前进到下一个光标位置            | Ctrl + Alt + →             | Cmd + Alt + →        | ★★★★☆  |

### 三、代码重构（优化代码结构）

| 功能描述                       | Windows 快捷键                  | Mac 快捷键                    | 优先级 |
| ------------------------------ | ------------------------------- | ----------------------------- | ------ |
| 重命名（类 / 方法 / 变量）     | Shift + F6                      | Shift + F6                    | ★★★★★  |
| 提取方法（将代码块封装为方法） | Ctrl + Alt + M                  | Cmd + Alt + M                 | ★★★★☆  |
| 提取变量（将表达式赋值为变量） | Ctrl + Alt + V                  | Cmd + Alt + V                 | ★★★★☆  |
| 提取常量                       | Ctrl + Alt + C                  | Cmd + Alt + C                 | ★★★☆☆  |
| 提取参数                       | Ctrl + Alt + P                  | Cmd + Alt + P                 | ★★★☆☆  |
| 移动代码块（上下移动行）       | Alt + ↑ / Alt + ↓               | Option + ↑ / Option + ↓       | ★★★★☆  |
| 复制代码块（上下复制行）       | Ctrl + Alt + ↑ / Ctrl + Alt + ↓ | Cmd + Alt + ↑ / Cmd + Alt + ↓ | ★★★☆☆  |
| 安全删除（检查引用后删除）     | Alt + Delete                    | Option + Delete               | ★★★☆☆  |

### 四、调试运行（排查代码错误）

| 功能描述                     | Windows 快捷键 | Mac 快捷键  | 优先级 |
| ---------------------------- | -------------- | ----------- | ------ |
| 启动调试                     | Shift + F9     | Shift + F9  | ★★★★☆  |
| 启动运行（不调试）           | Shift + F10    | Shift + F10 | ★★★★☆  |
| 断点开关（当前行）           | Ctrl + F8      | Cmd + F8    | ★★★★★  |
| 下一步（逐行执行，不进方法） | F8             | F8          | ★★★★★  |
| 进入方法（逐行执行，进方法） | F7             | F7          | ★★★★☆  |
| 强制进入方法（包括第三方库） | Alt + F7       | Option + F7 | ★★★☆☆  |
| 跳出当前方法                 | Shift + F8     | Shift + F8  | ★★★★☆  |
| 继续执行（到下一个断点）     | F9             | F9          | ★★★★★  |
| 查看变量值（调试时）         | Alt + F8       | Option + F8 | ★★★☆☆  |
| 停止调试                     | Ctrl + F2      | Cmd + F2    | ★★★☆☆  |

### 五、版本控制（Git/SVN 常用）

| 功能描述                 | Windows 快捷键      | Mac 快捷键    | 优先级 |
| ------------------------ | ------------------- | ------------- | ------ |
| 提交代码（Commit）       | Ctrl + K            | Cmd + K       | ★★★★☆  |
| 拉取代码（Pull）         | Ctrl + T            | Cmd + T       | ★★★★☆  |
| 查看提交记录             | Alt + 9（VCS 窗口） | Option + 9    | ★★★☆☆  |
| 比较当前文件与版本库差异 | Ctrl + D（VCS 中）  | Cmd + D       | ★★★☆☆  |
| 撤销未提交的修改         | Ctrl + Alt + Z      | Cmd + Alt + Z | ★★★☆☆  |
